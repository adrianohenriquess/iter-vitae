###### Responsabilidades:
###### Sistema de Gestão de Recursos Hídricos do Estado do Paraná
<p align="justify">
Neste projeto, implementei diversas funcionalidades para um sistema de gestão de recursos hídricos voltado para o estado do Paraná. Entre as principais atividades realizadas, destaco:
Cadastro de Outorga: Desenvolvimento do fluxo de cadastro para a concessão de uso de recursos hídricos e lançamento de efluentes.
Integração de Sistemas: Estabeleci a integração entre o sistema de autenticação do estado do Paraná e o sistema de gestão de recursos hídricos, garantindo maior segurança e facilidade de acesso.
Geração de Boletos: Integração com o sistema de geração de boletos da Caixa Econômica Federal, facilitando o processo de pagamento de taxas e serviços.
Cadastro de Usuários: Implementação do módulo de administração, permitindo o cadastro e gerenciamento de usuários no sistema.
Além dessas funcionalidades, contribuí para a implementação de várias outras características essenciais para o sistema de gestão de recursos hídricos.
</p>

###### Detalhes da arquitetura.
<p align="justify">
JSF, PrimeFaces, EJB 3.1, JPA, Arquillian, JUnit 4, Selenide, além das abordagens BDD (Behavior-Driven Development) e TDD (Test-Driven Development). O banco de dados utilizado foi o PostgreSQL, em um ambiente de execução WildFly 10.
</p>
